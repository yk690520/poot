#### **项目说明**

```
Poot 是一个安卓手机控制框架，底层使用了DOM解析安卓ui界面文件，控制设备需要连接数据线。
```
```
安装：
安装方式，在当前目录下（管理员）：
如果linux，mac：
python3 setup.py install
如果windows
python setup.py install
```
